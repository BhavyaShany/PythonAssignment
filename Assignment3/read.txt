Library Inventory Manager

A lightweight command-line tool to help manage books in a library.
It supports adding books, issuing and returning them, searching the catalog, and saving data in a JSON file.
The project follows Object-Oriented Programming principles and uses file handling, logging, and modular structure.

✨ Features

Add new books with title, author, and ISBN

Issue and return books

Search by title or ISBN

View complete catalog

Persistent storage using JSON

Proper exception handling

Logging using Python’s logging module

Clean project packaging with modules
